# BlockVote Backend API

This directory contains the backend REST API server for the BlockVote platform.

## Structure
- `src/index.js` - Main Express server
- `src/votingAbi.json` - Smart contract ABI
- `src/deployed_voting.json` - Deployed contract addresses

## Running the Server

```bash
npm install
npm start
```

The server will run on port 6001 by default.
